# Rails Application with React Sample
## Features

- no Asset Pipeline
- use JSX

## Requirements
This sample currently works with:

- Rails 5.2.x
- PostgreSQL
- yarn

## Usage

```
# in root dir
$ bundle install
$ yarn install
$ foreman start
```
